使用方法

1. 直接在HTML中使用

<img src="assets/icons/user/order.svg" alt="订单图标">

2. 作为背景图片使用

.order-icon {
    background-image: url('assets/icons/user/order.svg');
    background-size: contain;
    background-repeat: no-repeat;
    width: 24px;
    height: 24px;
}

3. 使用SVG内联

将SVG代码直接嵌入HTML中，可以通过CSS控制图标颜色：

<!-- 例如控制颜色的内联SVG -->
<svg class="icon-svg" width="24px" height="24px" viewBox="0 0 24 24">
    <!-- SVG内容 -->
</svg>

<style>
.icon-svg {
    fill: currentColor; /* 使图标继承文本颜色 */
}
</style>

4. 图标大小调整

所有图标默认为24px×24px，可通过CSS调整大小：

.small-icon {
    width: 16px;
    height: 16px;
}

.large-icon {
    width: 32px;
    height: 32px;
}
